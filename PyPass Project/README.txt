The PyPass project is a simple console application designed to help users keep their passwords secure. The main function of the project
is to encrypt text chosen by the user using a selected encryption method and save it to a file. The user can then use the same
encryption method to decrypt the text.

The project works as follows:

1. The user starts the encryption process by entering the text to be encrypted, the encryption method, the key value, and the file name.
2. The text is encrypted using the selected encryption method and saved to a file with the specified file name.
3. The user can later use the same encryption method and enter the file name to decrypt the encrypted text.
4. The decrypted text is printed to the console.
This project is a simple tool that can be used to help users keep their passwords secure.

The contents of the PyPass.py file can be modified and improved as desired. The .exe version of this code that has been converted to an
executable file can be used from the Dist folder.

TR

PyPass projesi, kullanıcıların şifrelerini güvende tutmalarına yardımcı olmak için tasarlanmış basit bir konsol
uygulamasıdır. Projenin temel işlevi, kullanıcının seçtiği bir şifreleme yöntemiyle şifrelediği metni bir dosyaya
kaydetmektir. Ardından, kullanıcının aynı şifreleme yöntemini kullanarak şifreli metni geri çözmesine izin verir.
Bu sayede kullanıcılar şifrelerini güvende tutabilirler.

Projenin çalışma mantığı şu şekildedir:

1. Kullanıcı şifrelenecek metni, şifreleme yöntemini, anahtar değerini ve dosya adını girdikten sonra şifrelemeyi başlatır.
2. Şifreleme yöntemi kullanılarak metin şifrelenir ve belirtilen dosya adı ile birlikte bir dosyaya kaydedilir.
3. Kullanıcı daha sonra dosyanın adını girdikten sonra şifreli metni geri çözmek için aynı şifreleme yöntemini kullanır.
4. Geri çözülen metin konsola yazdırılır.

Bu proje, kullanıcıların şifrelerini güvende tutmalarına yardımcı olmak için kullanılabilecek basit bir araçtır.

PyPass.py dosyasının içeriğini dilediğiniz gibi değiştirebilir ve geliştirebilirsiniz. Dist klasörünün içinde bu kodun .exe uzantılı
dosya haline getirilmiş versiyonunu kullanılabilir.
